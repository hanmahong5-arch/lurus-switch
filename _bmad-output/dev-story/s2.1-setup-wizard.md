# S2.1: First-Time Setup Wizard

**Epic**: E2 — Onboarding & Dashboard
**Points**: 5
**Status**: done
**Completed**: 2026-02-27
**Depends on**: S2.5 (DetectSystemProxy used in Step 3)

## Story

**As a** new user,
**I want** a guided setup wizard on first launch,
**So that** I can configure my tools and proxy without reading documentation.

## Implementation

### Modified Files (Go)
- `internal/appconfig/appconfig.go` — added `OnboardingCompleted bool` field to `AppSettings`

### New Files (Frontend)
- `frontend/src/components/SetupWizard.tsx` — ~280 lines, 4-step wizard

### Modified Files (Frontend)
- `frontend/src/App.tsx` — wizard gate: checks `onboardingCompleted`, shows wizard or main app
- `frontend/src/pages/SettingsPage.tsx` — added "Re-run Setup Wizard" button in data tab
- `frontend/wailsjs/go/models.ts` — added `onboardingCompleted` to `appconfig.AppSettings`
- `frontend/src/i18n/zh.json` — added `wizard.*` keys (~20 keys)
- `frontend/src/i18n/en.json` — added `wizard.*` keys (~20 keys)

## Wizard Steps

| Step | Content | Data Sources |
|------|---------|-------------|
| 0 — Welcome | App intro, language selector | i18n |
| 1 — Tools | Tool detection + "Install All" | `DetectAllTools()` |
| 2 — Proxy | Auto-detect + manual input | `DetectSystemProxy()`, `SaveProxySettings()` |
| 3 — Done | Summary + "Go to Dashboard" | Saves `onboardingCompleted: true` |
