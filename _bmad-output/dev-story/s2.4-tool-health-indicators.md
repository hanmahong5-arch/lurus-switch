## Health Check Logic

| Tool | Red | Yellow | Green |
|------|-----|--------|-------|
| claude | Config not found / invalid JSON | Both API_KEY and BASE_URL empty | Has at least one configured |
| codex | Config not found / invalid TOML | Model field empty | Model set |
| gemini | Config not found / invalid JSON | model.name missing | model.name present |
| picoclaw/nullclaw | Config not found / invalid JSON | api_base empty | api_base set |
