# S2.5: Proxy Auto-Detection

**Epic**: E2 — Onboarding & Dashboard
**Points**: 5
**Status**: done
**Completed**: 2026-02-27

## Story

**As a** Chinese developer,
**I want** the app to auto-detect my proxy settings,
**So that** I don't need to manually configure them every time.

## Implementation

### New Files
- `internal/proxydetect/proxydetect.go` — `DetectedProxy` type + `DetectAll()` orchestration with dedup
- `internal/proxydetect/env.go` — `detectEnvVars()`: checks HTTP_PROXY/HTTPS_PROXY/ALL_PROXY (+ lowercase)
- `internal/proxydetect/ports.go` — `detectCommonPorts()`: TCP probe 7890/7891/10808/10809/1080 (1s timeout)
- `internal/proxydetect/system_windows.go` — `detectSystemProxy()`: reads Windows registry `ProxyEnable`+`ProxyServer`
- `internal/proxydetect/system_other.go` — no-op on non-Windows
- `internal/proxydetect/proxydetect_test.go` — 7 test cases

### Modified Files
- `bindings_proxy.go` — added `DetectSystemProxy()` binding
- `frontend/wailsjs/go/main/App.d.ts` — added `DetectSystemProxy` declaration
- `frontend/wailsjs/go/main/App.js` — added `DetectSystemProxy` function
- `frontend/wailsjs/go/models.ts` — added `proxydetect.DetectedProxy` class

## Acceptance Criteria

- [x] Detect `HTTP_PROXY` / `HTTPS_PROXY` / `ALL_PROXY` env vars (+ lowercase variants)
- [x] Detect Windows system proxy via registry
- [x] Detect common proxy ports: Clash (7890/7891), V2Ray (10808/10809), SOCKS (1080)
- [x] Results deduplicated by URL
- [x] Non-Windows stub returns nil
- [x] Wails binding exposed as `DetectSystemProxy()`
- [x] 7 unit tests passing
